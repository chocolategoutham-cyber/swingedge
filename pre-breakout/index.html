How it works

Constructive bases near a pivot

A pre-breakout stock is one building a constructive base near a clearly defined resistance level. The replica preserves the same language around consolidation, volatility contraction, volume dry-up, and improving relative strength.

Rows stay below the breakout definition until price clears the pivot with enough participation versus its 50-day average.

How to use it

Filter first, validate second

Start with the strongest signal buckets, add leading RS and volume context, then inspect whether the chart looks orderly rather than chaotic.

Use the table as research input, not as automatic execution logic.

Frequently used terms

VCP, Stage 2, volume dry-up, relative strength, and breakout proximity are presented as public definitions while exact ranking formulas remain undisclosed.
